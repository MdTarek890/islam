Java Proggraming

Java is a widely-used programming language for coding web applications. It has been a popular choice among developers for over two decades, with millions of Java applications in use today. Java is a multi-platform, object-oriented, and network-centric language that can be used as a platform in itself. It is a fast, secure, reliable programming language for coding everything from mobile apps and enterprise software to big data applications and server-side technologies.

Java offers various tools to support automated editing, debugging, testing, deployment, and change management. These tools make Java programming time and cost-efficient.Java code can run on any underlying platform like Windows, Linux, iOS, or Android without rewriting. This makes it especially powerful in todays environment, where we want to run applications on multiple devices Users can download untrusted Java code over a network and run it in a secure environment in which it cannot do any harm. Untrusted code cannot infect the host system with a virus nor can it read or write files from the hard drive. The security levels and restrictions in Java are also highly configurable.Java is popular because it has been designed for ease of use. Some reasons developers continue to choose Java over other programming languages include: Java has been around for a long time, so many learning resources are available for new programmers.

Java 提供了各种工具来支持自动编辑、调试、测试、部署和变更管理。这些工具使 Java 编程变得省时且经济高效。Java 代码可以在任何底层平台（如 Windows、Linux、iOS 或 Android）上运行，无需重写。这使得它在当今的环境中特别强大，我们希望在多个设备上运行应用程序，用户可以通过网络下载不受信任的 Java 代码，并在安全的环境中运行它，而不会造成任何损害。不受信任的代码无法用病毒感染主机系统，也无法从硬盘驱动器读取或写入文件。 Java 中的安全级别和限制也是高度可配置的。Java 之所以流行，是因为它的设计易于使用。开发人员继续选择 Java 而不是其他编程语言的一些原因包括： Java 已经存在很长时间了，有很多学习资源可供新程序员使用。

All programming languages are a means to communicate with machines. Machine hardware only responds to electronic communication. High-level programming languages like Java act as a bridge between human language and hardware language. To use Java, a developer needs to understand two things:The Java Virtual Machine acts as an additional abstraction layer between the Java platform and the underlying machine hardware. Java source code can run only on those machines that have JVM installed on them. The answer to why the Java Virtual Machine is needed lies in the history of programming.

所有编程语言都是与机器通信的手段。机器硬件仅响应电子通信。像Java这样的高级编程语言充当人类语言和硬件语言之间的桥梁。要使用 Java，开发人员需要了解两件事： Java 虚拟机充当 Java 平台和底层机器硬件之间的附加抽象层。 Java源代码只能在安装了JVM的机器上运行。为什么需要 Java 虚拟机的答案在于编程的历史。
